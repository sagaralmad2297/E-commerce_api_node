E-commerce API project in NODEJS

--System Requirement
  -NODEJS
  -MySQL
  -NPM

  -setting up the project

  -clone the repository
  -go inside the cloned folder from the terminal/cmd
  -Run npm install
  -created a new file .env and configure the following enviroment variables
   .port=3000
  -Run the project
    Run `node server.js`
